### 立即执行函数里传window

```js
(function (window, undefined){
    
})(window)
```

* 减少作用域链查找，访问速度更快
* `undefined`是变量，可以重新赋值但无效，`null`是关键字不能赋值

### Object.defineProperty()取代直接修改原型链

```js
vue.prototype.$router = {}

Object.defineProperty(vue.prtotype, '$route', {
	set:function(){
		...
	}
})
```
