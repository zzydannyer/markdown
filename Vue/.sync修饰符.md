## **.sync修饰符**

父组件

```
<document 
    :title="doc.title" 
    :content="doc.content" 
    @update:title="doc.title = $event"
    @update:content="doc.content = $event"
 />
```

子组件

```
changeTitle(){
    this.$emit('update:title','new title')
}
changeContent(){
    this.$emit('update:content','new content')
}
```

使用.sync

```
<document 
    v-bind.sync="doc"
/>
```
