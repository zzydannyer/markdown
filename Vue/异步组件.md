## **异步组件**
* #### 不直接import
```
components:{
    List:() => import (/*webpackChunkName:"list"*/'../../List')
}
```
&emsp;&emsp;定义webpackChunkName => 下载异步组件的js名称 
* #### import写法
```
import Loading from './loading/'
import Error from './error /'

const AsyncList=() => ({
    component:import(/*webpackChunkName:"list"*/'./List'),
    loading:Loading,
    error:Error,
    delay:200,       //延迟加载
    timeout:3000     //加载失败显示Error组件
})

components:{
   AsyncList
}
```
